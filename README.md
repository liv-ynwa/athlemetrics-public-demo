# Athlemetrics Platform Public Demo

Welcome to the Athlemetrics public demo platform! Athlemetrics is a comprehensive platform focused on football data analysis and AI models, designed for football enthusiasts, data analysts, coaches, and academic researchers. This directory provides a public demo version of Athlemetrics, showcasing the platform’s core features and technical achievements. Please note that this demo does not include any backend source code, model weights, or sensitive data, and is intended for reference and experience only.

## Core Features and Online Experience

Athlemetrics offers three key online modules to help users gain deeper insights and applications of football data:

- **Rating Demo**  
  Evaluates player performance in matches using data-driven rating models, enabling users to intuitively understand player abilities.

- **Player Similarity Explorer**  
  Uses multi-dimensional data analysis to discover players with styles and performances similar to a target player, supporting scouting and analysis work.

- **Tactical Role Analyzer**  
  Analyzes player roles within different tactical systems, assisting coaches and analysts in tactical planning and adjustments.

You can directly access these online experiences via the following links:

- [Rating Demo](https://platform.athlemetrics.com/rating-demo/)  
- [Player Similarity Explorer](https://platform.athlemetrics.com/player-similarity/)  
- [Tactical Role Analyzer](https://platform.athlemetrics.com/tactical-analyzer/)

## Local Run (Optional)

If you want to preview the demo pages locally, you can open the `index.html` file directly or start a local server using the following command:

```bash
python3 -m http.server 8000
```

Then access [http://localhost:8000/](http://localhost:8000/) in your browser.
